# 🧜‍♂️ Mermaid Editor v0.1

這是一個基於 AI 協作開發的輕量化 **Mermaid 語法即時預覽工具**。透過簡單的文字代碼，即可將複雜的邏輯、流程與計畫，瞬間轉化為直觀的視覺圖表。

> **「一眼看透邏輯，勝過千言萬語。」** —— 專為提升組織規劃能力而生。

## 🚀 核心功能
- **即時預覽 (Real-time Rendering)**：左邊輸入代碼，右邊立即出圖，支援 Mermaid 全系列圖表。
- **互動式介面**：支援圖表放大、縮小、重置視圖。
- **深色模式切換**：完美適應各種工作環境視覺需求。
- **零成本部署**：基於 GitHub Pages，無需後端，純前端運算，隱私安全。

## 💡 為什麼需要它？
- **對上班族**：快速產出專業流程圖、甘特圖或使用者旅程圖，提升專案溝通效率。
- **對長輩/初學者**：將混亂的待辦事項（如：申請護照流程、每日行程）轉化為路徑圖，降低認知負擔。
- **對 AI 使用者**：Gemini/ChatGPT 生成的 Mermaid 代碼，直接貼進來就能看，不用再付費買專業軟體。

## 🛠️ 使用技術
- [Mermaid.js](https://mermaid.js.org/) - 開源圖表渲染引擎
- HTML5 / Tailwind CSS - 介面美化
- AI 協作開發 (Qwen 3.6 Plus 輔助設計)

## 📖 如何使用
1. 打開網址：`https://yutbfrec-ops.github.io/Mermaid_Editor/`
2. 在代碼視窗中貼入或撰寫 Mermaid 語法（例如 `flowchart TD`）。
3. 點擊「重新渲染」或觀看即時變化。
